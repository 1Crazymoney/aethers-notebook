Aether's Notebook
=========

About
-----

Aether's Notebook is a logging application for wireless (3G/WiFi) connectivity for Android
mobile phones.  It has been developed for the [Horizon][] infrastructure project.

[Horizon]: http://www.horizon.ac.uk
